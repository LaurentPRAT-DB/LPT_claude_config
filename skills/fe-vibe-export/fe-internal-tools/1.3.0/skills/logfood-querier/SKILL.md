# Internal Data Querier Skill

Queries Databricks internal data and metrics to understand consumption, feature usage, account information, and more. 

### Instructions
1) Use the databricks-query skill to execute queries
2) For GTM data (sales, forecast, consumption, usage, accounts, etc) refer to resources/GTM_DATA.md to identify candidate tables and queries
3) For Salesforce data (use cases, opportunities, blockers, escalations, ASQs, etc) use the `salesforce-actions` skill instead
