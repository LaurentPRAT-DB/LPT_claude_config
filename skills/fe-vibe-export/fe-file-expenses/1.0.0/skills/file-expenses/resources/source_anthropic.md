# Anthropic (Claude Pro) Receipt Download

## Overview
Download Claude Pro subscription invoices from claude.ai for expense reporting.

## Authentication
- URL: https://claude.ai/settings/billing
- Login: Google SSO or email/password
- Note: May encounter Cloudflare verification

## Navigation Steps (to be documented)

### 1. Access Billing Settings
```
https://claude.ai/settings/billing
```

### 2. View Invoice History
- Navigate to billing section
- Look for "Invoice History" or "Billing History"
- Find invoices in the relevant date range

### 3. Download Invoice PDF
- Click on invoice to view details
- Download as PDF

## Expense Category
**Career Development** - Use this category in Emburse

## Receipt Details
Claude Pro invoices typically include:
- Subscription period
- Amount ($20/month for Pro)
- Payment method
- Invoice date and number

## Email Upload to Emburse
After downloading, email receipts to: `receipt@ca1.chromeriver.com`
- Subject: Amount (e.g., "20.00 USD")
- Attach the PDF invoice

## Notes
- Monthly subscription (~$20/month or ~$200/year)
- Invoices sent via email as well - check inbox for "Anthropic" or "Claude"
- May also appear as API usage if using console.anthropic.com
